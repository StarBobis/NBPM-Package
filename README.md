# NBPM-Package

3Dmigoto Preconfigured For Certain Action Game Mobile Version.

- F6 to switch between mod open or close.
- Ctrl + U to hide or show GUI for take screenshot.

# Existed bug

It's under test so not function on all scene, now if you modify cloth ,the physics of cloth will lose.

# Manually Check Used

This config use manually check instead of global check to:

- Prevent FPS decrease.
- Avoid Game Crash due to global check.
- Avoid shader lag type FPS decrease with global check.

Don't forget to update VSCheck list and generate VSCheck in your SSMT's WorkPage after model extract when you make mod.
